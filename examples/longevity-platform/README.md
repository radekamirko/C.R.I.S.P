# Example: Longevity Intelligence Platform

> **Note:** This example was produced with CRISP v1.0 — before the methodology was updated. File naming conventions and some phase outputs differ from the current standard. For a full example using the current CRISP, see [znam-coveka](../znam-coveka/).

It's kept here as a reference point — useful for seeing how the framework evolved.

---

**Project:** Longevity Intelligence Platform — a bootstrapped product helping users understand and act on longevity research.

**Tested by:** Mirko Radeka

---

## What's different from current CRISP

- Files use `phase1-`, `phase2-` prefixes instead of living in `docs/`
- Missing Phase C outputs: market research, SWOT, value proposition canvas
- Pre-dates UX discovery, HVLE live conversation, NFR elicitation, and sprint-specific open questions
