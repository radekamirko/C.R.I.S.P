# Example: Znam Coveka

A real CRISP engagement, run start to finish.

**Project:** Znam Coveka — a match-based platform connecting homeowners with vetted tradies in Serbia. No bidding. Homeowners get matched tradies with an AI-generated explanation of why each one fits their job. Tradies pay for access; homeowners don't.

**Tested by:** Mirko Radeka
